# Orbi-App-Financeiro
Entrega da tarefa prática do curso utilizando a ferramenta Lovable.

O objetivo foi criar um aplicativo de organização financeira que finalmente fosse fácil de usar. 
A marioria das pessoas baixa apps de organização financeira e acaba não usando porque são confusos, demorados de inserir os dados, tem muitos recursos e acabam sendo pouco práticos para o uso do dia a dia. 
A ideia foi fazer um app conversacional, onde a pessoa possa inserir os dados por comando de voz, sem precisar digitar nada. O aplicativo transformará o comando em dado e colocará os dados 
no sistema, classificando como receita ou despesa e em subcategorias dentro de cada uma delas. 

Obs: Fiz duas tentativas de integrar um design criado no Figma mas o Lovable não consegue incorporar o estilo.
